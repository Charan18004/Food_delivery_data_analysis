# Food Delivery Data Analysis

This project combines multiple data sources (CSV, JSON, and SQL) to build a unified food delivery analytics dataset and perform business insights using Python and Pandas.

## Dataset Sources
- orders.csv → order transactions
- users.json → customer details
- restaurants.sql → restaurant information

## Tasks Performed
- Loaded data from CSV, JSON, and SQL
- Executed LEFT JOIN operations to merge datasets
- Created final consolidated dataset
- Calculated highest revenue city among Gold members
- Identified cuisine with highest average order value

## Tools Used
Python, Pandas, SQLite, Jupyter Notebook
